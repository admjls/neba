NEBA Release 3.3.0

Oct232014

Release 3.3.0

  • Posted by {% if page.twitterAuthor%}{% endif %}Olaf Otto{% if page.twitterAuthor %}{% endif %}

Since 3.2.0, we've added new features to the annotations @Children (resolveBelowEveryChild), @Reference (append) and @ResourceParam (defaultValue) to allow even more efficient declarative use of the resource hierarchy. Furthermore, this release updates the Spring version shipped with the delivery package to 3.2.11, thus you benefit from all fixes and improvements added since 3.2.9 (shipped with NEBA 3.2.0).

Along with this release, the documentation - most importantly the NEBA user guide - was updated and improved.

Improvements

#10 Add support for a default value to the @ResourceParam tag
#11 Support declarative sub-path resolution in @Reference and @Children
#13 Update documentation with regard to changes in neba-11
#14 Update to Spring 3.2.11, the current 3.2.x Spring version
